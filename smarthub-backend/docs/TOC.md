# Table des contenus — Documentation SmartHub

Ce dossier `docs/` contient la documentation détaillée du backend SmartHub. Utilise ce fichier comme point d'entrée.

Fichiers disponibles :

- `controllers.md` — description des controllers et signatures d'API.
- `services.md` — description des services (logique métier) avec méthodes publiques importantes.
- `repositories.md` — interfaces JPA et méthodes dérivées.
- `models.md` — entités JPA, champs principaux et relations (ERD simplifié).
- `dtos.md` — DTOs request/response et règles de validation.
- `config.md` — configuration (Security, JWT, CORS, RestTemplate, Ollama).
- `agents.md` — architecture et rôle des agents métier (orchestrateurs, trackers).
- `rag.md` — architecture Retrieval-Augmented Generation (RAG), embedding flow, indexation.
- `diagrams.md` — diagrammes Mermaid pour architecture, classes, use-cases, ERD, agents, RAG.
- `packages.md` — index des packages et liens vers la documentation par classe (docs/classes/)
- `classes/` — documentation générée par classe (fichiers Markdown par classe)

Guide : Lis `TOC.md` puis ouvre `packages.md` pour naviguer la documentation par package ou accède directement à `docs/classes/` pour fichiers par classe.
